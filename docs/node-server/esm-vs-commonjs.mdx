# ESM vs COMMONJS

## ESM Vs COMMONJS

| ESM                                                                         |                                        COMMONJS                                         |
| --------------------------------------------------------------------------- | :-------------------------------------------------------------------------------------: |
| ECMAScript or ES Modules (ESM) rely on the **import** and **export** syntax | CommonJS (CJS) approach was default approach it uses **require** and **module.exports** |
| Need to add **type:module** in **package.json**                             |                    Need to add **type:commonjs** in **package.json**                    |
| Asynchronous                                                                |                                       Synchronous                                       |
| caching is not done                                                         |                                     caching is done                                     |
