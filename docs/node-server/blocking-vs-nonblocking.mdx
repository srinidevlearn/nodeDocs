# Blocking vs Non-Blocking

## Synchronous Vs Asynchronous

| Blocking                                                                   |                                          Non-Blocking                                          |
| -------------------------------------------------------------------------- | :--------------------------------------------------------------------------------------------: |
| Blocking is eventloop must wait until a non-JavaScript operation completes | Non-Blocking is eventloop gets response using callback when non-JavaScript operation completes |
| works synchronously                                                        |                                      works asynchronously                                      |
